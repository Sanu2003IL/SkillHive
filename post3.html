📝 Arithmetic Operators

a = 10
b = 3
print(a + b)   # 13
print(a - b)   # 7
print(a * b)   # 30
print(a / b)   # 3.33

📝 Comparison Operators

x = 5
y = 8
print(x > y)   # False
print(x == 5)  # True

📝 Assignment Operators

x = 10
x += 5   # same as x = x + 5
print(x) # 15
