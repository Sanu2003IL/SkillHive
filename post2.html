📝 What are Variables?

Variables are containers for storing data values.

x = 10
name = "Alice"
pi = 3.14

🔑 Data Types

Common data types in Python:

  • int → whole numbers
  • float → decimal numbers
  • str → text
  • bool → True/False

✅ Example

age = 20
height = 5.9
student = True
print(age, height, student)
