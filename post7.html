📅 Day 6 – Lists in Python

A list is a collection of ordered, mutable (changeable), and heterogeneous (can store different types) elements. Lists are written with square brackets [ ].

1. Creating Lists


# Empty list
my_list = []

# List of numbers
nums = [1, 2, 3, 4, 5]

# List of strings
fruits = ["apple", "banana", "cherry"]

# Mixed data types
mixed = [1, "hello", 3.14, True]

# Nested list
nested = [[1, 2], [3, 4]]

2. Accessing Elements


fruits = ["apple", "banana", "cherry"]

print(fruits[0])    # apple (first element)
print(fruits[-1])   # cherry (last element)
print(fruits[1:])   # ['banana', 'cherry']

3. Modifying Lists


fruits = ["apple", "banana", "cherry"]

# Change element
fruits[1] = "mango"
print(fruits)   # ['apple', 'mango', 'cherry']

# Add element
fruits.append("orange")
print(fruits)

# Insert at index
fruits.insert(1, "grape")
print(fruits)

# Remove element
fruits.remove("mango")
print(fruits)

# Pop element (last by default)
print(fruits.pop())   # removes last
print(fruits)

# Delete element by index
del fruits[0]
print(fruits)

# Clear list
fruits.clear()
print(fruits)

4. List Operations


list1 = [1, 2, 3]
list2 = [4, 5, 6]

print(list1 + list2)   # Concatenation
print(list1 * 2)       # Repetition
print(2 in list1)      # Membership -> True
print(10 not in list1) # True

5. List Functions


numbers = [10, 20, 30, 40, 50]

print(len(numbers))    # 5
print(max(numbers))    # 50
print(min(numbers))    # 10
print(sum(numbers))    # 150

6. List Methods


nums = [3, 1, 4, 1, 5, 9]

nums.sort()
print(nums)       # [1, 1, 3, 4, 5, 9]

nums.reverse()
print(nums)       # [9, 5, 4, 3, 1, 1]

print(nums.count(1))  # 2
print(nums.index(4))  # 2

7. Iterating Over Lists


fruits = ["apple", "banana", "cherry"]

for fruit in fruits:
    print(fruit)

# With index
for i in range(len(fruits)):
    print(i, fruits[i])

8. List Comprehension


# Squares of numbers 1 to 5
squares = [x**2 for x in range(1, 6)]
print(squares)  # [1, 4, 9, 16, 25]

# Even numbers
evens = [x for x in range(10) if x % 2 == 0]
print(evens)  # [0, 2, 4, 6, 8]

9. Nested Lists (Matrix Example)


matrix = [[1, 2, 3], [4, 5, 6], [7, 8, 9]]

print(matrix[0][1])  # 2

for row in matrix:
    for col in row:
        print(col, end=" ")

👉 Day 6 Summary

  • Lists are ordered, mutable collections.

  • Learned functions, methods, and list comprehensions.
