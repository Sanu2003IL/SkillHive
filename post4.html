🔹 1. What is Control Flow?

Control flow decides which part of the program runs and when. Normally, Python runs code top to bottom, but with if/else we make decisions.

Example: If it rains → take an umbrella. Else → wear sunglasses.

🔹 2. if Statement

age = 20

if age >= 18:
    print("You are an adult.")

🔹 3. if-else Statement

age = 16

if age >= 18:
    print("You can vote.")
else:
    print("You cannot vote yet.")

🔹 4. if-elif-else (Multiple Conditions)

marks = 75

if marks >= 90:
    print("Grade: A")
elif marks >= 75:
    print("Grade: B")
elif marks >= 50:
    print("Grade: C")
else:
    print("Grade: F")

🔹 5. Nested if Statements

age = 20
citizen = True

if age >= 18:
    if citizen:
        print("You can vote in elections.")
    else:
        print("You must be a citizen to vote.")
else:
    print("You are under 18.")

🔹 6. Comparison Operators

  • == → Equal to
  • != → Not equal to
  • > → Greater than
  • < → Less than
  • >= → Greater or equal
  • <= → Less or equal

🔹 7. Logical Operators

  • and → Both must be True
  • or → At least one True
  • not → Negates condition
age = 22
student = True

if age > 18 and student:
    print("Adult student.")

🔹 8. Shorthand if

x = 10
if x > 5: print("x is greater than 5")

🔹 9. Ternary Operator

age = 17
status = "Adult" if age >= 18 else "Minor"
print(status)  # Minor

🔹 10. Real-Life Examples

✅ Even or Odd

num = 7
if num % 2 == 0:
    print("Even number")
else:
    print("Odd number")

✅ Login System

username = "admin"
password = "1234"

if username == "admin" and password == "1234":
    print("Login Successful!")
else:
    print("Invalid credentials")

✅ Traffic Signal

signal = "red"

if signal == "green":
    print("Go")
elif signal == "yellow":
    print("Wait")
else:
    print("Stop")
