📅 Day 5 – Strings in Python (Detailed Study)

Strings are one of the most commonly used data types in Python. They are sequences of characters enclosed in single quotes (‘ ’), double quotes (“ ”), or triple quotes (‘’’ or “””).

1. Creating Strings


# single quotes
s1 = 'Hello'

# double quotes
s2 = "World"

# triple quotes (used for multi-line strings or docstrings)
s3 = '''This is 
a multi-line 
string.'''

print(s1)
print(s2)
print(s3)

Output:

Hello
World
This is 
a multi-line 
string.

2. String Indexing and Slicing

Strings are ordered sequences, so we can access characters using indexes.


text = "Python"

# Indexing
print(text[0])   # First character -> P
print(text[-1])  # Last character -> n

# Slicing
print(text[0:4])   # Pyth
print(text[:3])    # Pyt
print(text[2:])    # thon
print(text[::-1])  # Reverse -> nohtyP

3. String Operations

🔹 Concatenation


a = "Hello"
b = "World"
print(a + " " + b)  # Hello World

🔹 Repetition


word = "Hi"
print(word * 3)  # HiHiHi

🔹 Membership


text = "Python"
print("Py" in text)        # True
print("Java" not in text)  # True

4. Common String Functions


msg = "   Python Programming   "

print(len(msg))         # Length -> 24
print(msg.upper())      # Uppercase
print(msg.lower())      # Lowercase
print(msg.strip())      # Removes extra spaces
print(msg.replace("Python", "Java"))  # Replace
print(msg.split())      # Split -> ['Python', 'Programming']
print(msg.find("Pro"))  # Find substring -> 9

5. String Formatting

🔹 Using f-strings


name = "Alice"
age = 22
print(f"My name is {name} and I am {age} years old.")

🔹 Using .format()


print("My name is {} and I am {} years old.".format("Bob", 25))

🔹 Old-style % formatting


print("My name is %s and I am %d years old." % ("Charlie", 30))

6. Escape Characters


print("Hello\nWorld")   # New line
print("Hello\tWorld")   # Tab space
print("She said \"Python is fun\"")  # Quotes inside string

Output:

Hello
World
Hello    World
She said "Python is fun"

7. String Iteration (Looping)


text = "Python"
for ch in text:
    print(ch)
