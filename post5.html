2. String Indexing and Slicing

Strings are ordered sequences, so we can access characters using indexes.


text = "Python"

# Indexing
print(text[0])   # First character -> P
print(text[-1])  # Last character -> n

# Slicing
print(text[0:4])   # Pyth
print(text[:3])    # Pyt
print(text[2:])    # thon
print(text[::-1])  # Reverse -> nohtyP

4. Common String Functions


msg = "   Python Programming   "

print(len(msg))         # Length -> 24
print(msg.upper())      # Uppercase
print(msg.lower())      # Lowercase
print(msg.strip())      # Removes extra spaces
print(msg.replace("Python", "Java"))  # Replace
print(msg.split())      # Split -> ['Python', 'Programming']
print(msg.find("Pro"))  # Find substring -> 9
