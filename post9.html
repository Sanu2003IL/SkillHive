📘 Day 9 – Loops (Advanced Concepts)

Today we’ll explore nested loops, loop control statements, and some practical applications of loops in Python. 🚀

1. Quick Recap of Loops

A loop is used to repeat a block of code multiple times.

  • for loop → used when number of iterations is known.
  • while loop → used when repetitions depend on a condition.

for i in range(5):
    print("Hello", i)
    

count = 0
while count < 5:
    print("Count is:", count)
    count += 1

2. Nested Loops

A loop inside another loop. Example: Multiplication table.


for i in range(1, 4):   # Outer loop
    for j in range(1, 4):   # Inner loop
        print(i, "*", j, "=", i*j)
    print("----")

Output:


1 * 1 = 1
1 * 2 = 2
1 * 3 = 3
----
2 * 1 = 2
2 * 2 = 4
2 * 3 = 6
----
3 * 1 = 3
3 * 2 = 6
3 * 3 = 9
----

3. Loop Control Statements

(a) break – exit the loop immediately


for i in range(10):
    if i == 5:
        break
    print(i)

(b) continue – skip current iteration


for i in range(10):
    if i % 2 == 0:
        continue
    print(i)

(c) pass – does nothing (placeholder)


for i in range(5):
    if i == 3:
        pass   # to be implemented later
    else:
        print("Number:", i)

4. Using else with Loops

The else block executes only if loop finishes normally (not interrupted by break).


for i in range(5):
    print(i)
else:
    print("Loop completed successfully!")
    

for i in range(5):
    if i == 3:
        break
    print(i)
else:
    print("This will not run")

5. Practical Problems with Loops

🔹 Factorial of a Number


num = 5
factorial = 1

for i in range(1, num + 1):
    factorial *= i

print("Factorial of", num, "is", factorial)

🔹 Prime Number Check


num = 29
is_prime = True

for i in range(2, num):
    if num % i == 0:
        is_prime = False
        break

if is_prime:
    print(num, "is a prime number")
else:
    print(num, "is not a prime number")

🔹 Pattern Printing (Pyramid)


rows = 5
for i in range(1, rows+1):
    print("*" * i)

Output:


*
**
***
****
*****

Inverted pyramid:


rows = 5
for i in range(rows, 0, -1):
    print("*" * i)

🔹 Guess the Number Game


secret = 7
while True:
    guess = int(input("Enter your guess (1-10): "))
    if guess == secret:
        print("Correct! 🎉")
        break
    else:
        print("Wrong guess, try again!")
